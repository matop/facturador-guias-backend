# Triage Labels

Los skills hablan en términos de cinco roles canónicos de triage. Este archivo mapea esos roles a los strings de label reales usados en el tracker de este repo (GitHub Issues de `matop/facturador-guias-backend`).

| Label en mattpocock/skills | Label en nuestro tracker | Significado                               |
| --------------------------- | ------------------------- | ------------------------------------------ |
| `needs-triage`               | `needs-triage`             | El maintainer necesita evaluar este issue |
| `needs-info`                 | `needs-info`               | Esperando más información del reporter    |
| `ready-for-agent`            | `ready-for-agent`          | Totalmente especificado, listo para un agente AFK |
| `ready-for-human`            | `ready-for-human`          | Requiere implementación humana            |
| `wontfix`                    | `wontfix`                  | No se va a accionar                       |

Cuando un skill menciona un rol (ej. "aplicar el label de triage AFK-ready"), usar el string de label correspondiente de esta tabla.
