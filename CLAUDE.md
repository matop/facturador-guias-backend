## Agent skills

### Issue tracker

Issues y specs viven como GitHub Issues en `matop/facturador-guias-backend` (vía `gh`); PRs no son superficie de triage. Ver `docs/agents/issue-tracker.md`.

### Triage labels

Los 5 labels canónicos (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`) ya existen en el repo con esos mismos nombres. Ver `docs/agents/triage-labels.md`.

### Domain docs

Repo single-context: `CONTEXT.md` en la raíz + `docs/adr/` para decisiones arquitectónicas pasadas. Ver `docs/agents/domain.md`.
